GoblinPouch - Android Budgeting Application

Mobile Application Development

Description

Goblin Pouch is a collaborative Android budgeting application designed to make financial management simple, transparent, and engaging. Built in Kotlin with Firebase integration, the app allows users to track expenses in real time, create shared budgets with friends or family, and visualize spending through interactive dashboards and graphs.

To encourage healthy financial habits, Goblin Pouch introduces gamification: users earn points for sticking to their budgets, which can be redeemed for avatars in a reward shop. Shared budgets include expiring invite codes and tagged expenses, ensuring accountability and transparency. The app also supports receipt attachments, secure authentication, and persistent sessions, making it a comprehensive tool for everyday budgeting.

Technologies Used
  • Kotlin + XML -
    Core language and UI layout for Android development
  • Firebase Authentication -
    Secure sign-up, login, Google sign-in, and password reset
  • Firebase Realtime Database -
    Persistent storage and real-time synchronization of expenses and budgets
  • Android Navigation Components -
    Side navigation drawer and bottom navigation bar for modern UX
  • Graphical Libraries -
    Dynamic charts and progress indicators for budget visualization
Purpose

The project was developed as a group assignment to demonstrate practical Android development skills and teamwork in building a real-world financial application.

  • For Users:
    Goblin Pouch provides a secure, engaging way to manage personal and shared budgets, track spending habits, and stay motivated through gamification.
  • For developers:
    The project highlights integration of Firebase services, responsive UI/UX design, and collaborative development practices.
  • Learning outcome:
    Building Goblin Pouch reinforced skills in mobile app architecture, authentication, and real-time data handling, while showcasing how gamification can be applied to financial management.
